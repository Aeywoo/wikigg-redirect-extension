Redirect to wiki.gg becomes wiki.gg Redirect

Additional action is required to continue receiving updates.

The extension is now considered Freedom Games's (the company behind wiki.gg) first-party project. Because of the limitations of Microsoft Edge's Add-ons Portal, this required publishing the extension again under their developer account.

To keep up to date with future changes, updates, and wiki support, please install the new, first-party version found here. Afterwards uninstall this version.

"Redirect to wiki.gg" will continue to work, but will not receive any future code or wiki support updates.

This message is being shown to you, because you're a user of Microsoft Edge and have installed Redirect to wiki.gg from the Edge Add-ons Portal before the publisher has been changed.
